Custom PCB to control DC motors using [LMD18200](https://www.ti.com/lit/ds/symlink/lmd18200.pdf)

MCU: [STM32 NUCLEO-G431KB](https://www.st.com/en/evaluation-tools/nucleo-g431kb.html), code is here https://github.com/VictorDubois/Motor-board-STM32-CAN/tree/commit-all

Used by Krabi V1
Krabi V2 uses https://github.com/VictorDubois/CanMotorBoard
