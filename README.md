Custom PCB, used by KrabiV2.

Drives two brushless motors C610/M2006, controlled with dedicated encoders wheels, and 4 simple transistor outputs.

Driven by a [STM32 NUCLEO-G431KB](https://www.st.com/en/evaluation-tools/nucleo-g431kb.html)

[The code is here](https://github.com/VictorDubois/Motor-board-STM32-CAN)
